### 像数 1, 2, 3 一样容易

<CodeGroup>
  <CodeGroupItem title="YARN" active>

```bash
# 创建模板
yarn create @webank/fes-app myapp

# 安装依赖
yarn 

# 运行
yarn dev
```

  </CodeGroupItem>

  <CodeGroupItem title="NPM">

```bash
# 创建模板
npx @webank/create-fes-app myapp

# 安装依赖
npm install 

# 运行
npm run dev
```

  </CodeGroupItem>
</CodeGroup>
